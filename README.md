# CharFiller

This app fills a file with the \`a\` character up to a certain given file size.

INFO:

Due to the fact that the app is not multithreaded, it is currently very slow when writing to files.

## Usage

`-F [Filename]`

`-S [Size]`

`-T [Thread Number]`
