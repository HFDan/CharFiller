# CharFiller

This app fills a file with a random character up to a certain given file size.

## Usage

`-F` `[Filename]`

`-S` `[Size]`

`-T` `[Thread Number]`

`-NR` Non-random

`-C` `[Character to fill with]` <- Only usable with `-NR`
